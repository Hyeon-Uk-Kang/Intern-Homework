   작성법은 int [] num={1,2,3,4,5}; 
   라고 한다면 1,2,3,4,5가 순서대로 num이라는 배열에 담긴다.

## 예외 처리

   예외(Exception)은 프로그래머가 직접 예측 가능하여 막을 수 있는 오류이다. 
   
   try, catch, finally를 사용해 예외를 처리하거나 메소드를 호출한 곳으로 던질 수 있다.
